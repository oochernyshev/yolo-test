# yolo-test
Testing of yolo models

## Setup (macOS/Linux)

```bash
cd /Users/oleg/Projects/yolo-test
chmod +x setup.sh
./setup.sh
```

## Run

```bash
cd /Users/oleg/Projects/yolo-test
source .venv/bin/activate
python -m streamlit run app.py
```

Or use the helper script:

```bash
cd /Users/oleg/Projects/yolo-test
chmod +x start.sh
./start.sh
```
